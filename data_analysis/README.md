# Data Analysis Projects

This folder contains beginner to intermediate level data analysis projects using Python.

## Projects

### 1. CSV Data Analyzer
- Reads CSV files
- Calculates total, average, minimum, and maximum values
- Useful for simple data reporting

Script: `csv_data_analyzer.py`

---

### 2. Student Marks Analyzer
- Takes student marks as input
- Calculates total, average
- Assigns grade
- Identifies highest and lowest marks

Script: `student_marks_analyzer.py`

---

## How to Run

1. Open terminal
2. Go to the data_analysis folder

Example:
cd data_analysis

3. Run:

python csv_data_analyzer.py
# or
python student_marks_analyzer.py
