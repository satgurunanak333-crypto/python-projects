# Automation Projects (Python)

This folder contains Python scripts for automating common tasks.

## Projects

### 1. File Organizer
Organizes files in a folder into subfolders based on file type (Images, Documents, Videos, Others).

**Script:** `files_organizer.py`  
**Use case:** Clean messy folders automatically.

### 2. Bulk File Renamer
Renames multiple files in a folder at once using prefix/suffix.

**Script:** `bulk_file_renamer.py`  
**Use case:** Rename many files quickly (e.g., photos, documents).
